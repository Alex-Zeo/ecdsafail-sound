ecdsa.fail · SOUND

Honest fault-tolerant resource cost of a reversible secp256k1 point-addition circuit — ungameable validation.

The original ecdsa.fail grader seeds its 9024 test inputs from a Fiat-Shamir hash of your own op stream, so a contestant can re-roll which inputs are tested for free (the "tail-nonce hunt") and pass a circuit that is wrong on reachable inputs. This fork seeds test inputs from a grader-controlled source independent of the ops, so only a genuinely-correct circuit validates (0 classical / 0 phase / 0 ancilla across many independent seeds — the standard Google used a zero-knowledge proof for). The original board's sub-1300-qubit "circuits" fail here. Score = peak qubits × avg-executed Toffoli, lower is better.
